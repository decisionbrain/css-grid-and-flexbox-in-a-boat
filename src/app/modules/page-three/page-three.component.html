Grid fixed

**Fixed size contraints is very easy do define** ```css .two-line-grid-template {{ '{' }} display: grid; grid-template-rows: 400px 150px; grid-template-columns: 150px 300px 16px 300px 150px; } .one-line-grid-template {{ '{' }} display: grid; grid-template: 400px 150px / 150px 300px 16px 300px 150px; } ``` **The content can be aligned** ```css .sail.left {{ '{' }} justify-self: end; } ``` **And there are goodies too** ```css .center-old-school {{ '{' }} align-self: center; justify-self: center; } .center {{ '{' }} place-self: center; } ```
